Zmei Generator
============================================

   Zmei Generator got it's name after fabulous hero Змей Горыныч (zmey gorynych)
   That was, if it ever been, three head dragon throwing fire from his mouths.

   https://en.wikipedia.org/wiki/Zmei_(Russian)

Zmei generator is a special markup language that allows to describe `Django <https://www.djangoproject.com/>`_ application in a very
compact way and generate most of application code in an automatic way.

Crud
---------------------

@crud is a page annotation, that deserve a separate section in documentation
as it is very powerful and feature rich.

CRUD is software development pattern, that refers to very common way how
model data is handled in web applications `CRUD <https://en.wikipedia.org/wiki/Create,_read,_update_and_delete>`_

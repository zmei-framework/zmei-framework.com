Col file structure¶

Note

It is highly recommended to install Zmei PyCharm plugin, if you are using PyCharm. It adds handy syntax highlighting as on screenshot bellow.

Typical col file have Application annotations, pages, models. Order is important!

Also in .col file you can use python Code imports. It’s separate topic, but it worth to mention that there is such feature in .col files.
